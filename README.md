# 📚 중학생 영단어 게임

이 프로젝트는 중학생 수준의 영어 단어를 재미있게 익히도록 만든 Streamlit 게임입니다.

- 한국어 뜻을 보고 영어 단어를 맞힙니다.
- 정답/오답 피드백을 바로 확인할 수 있습니다.
- 점수와 문제 수를 실시간으로 볼 수 있습니다.

### 실행 방법

Prerequisite: install `uv` if you don't already have it.

```
$ curl -LsSf https://astral.sh/uv/install.sh | sh
```

1. 의존성 설치

   ```
   $ uv sync
   ```

2. 앱 실행

   ```
   $ uv run streamlit run streamlit_app.py
   ```
